## Data Safety Audit

- Products, variants, prices, URLs, availability, add-to-cart, cart, and checkout remain powered by Shopify objects.
- `assets/products.js` was not activated or imported.
- Old localStorage cart, custom checkout, Google Apps Script order submission, and order tracking were not used.
- Product image fallback remains scoped to missing Shopify product media only.

## Main Visual Risks Before Repair

- Empty block-only homepage sections could produce black gaps.
- Product and category density did not match original static homepage.
- Static home-specific CSS had selector drift: `body[data-page="home"]` vs Shopify `body[data-template="index"]`.
- Footer structure lacked original support bars and premium footer rhythm.
